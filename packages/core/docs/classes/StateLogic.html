    Class StateLogic

    Base class for entity behavior. Extend to add custom logic.

    Example

    class MoveRight extends Logic<PIXI.Sprite> {
      speed = 100;
      update(entity, dt) { entity.x += this.speed * dt; }
    }

    sprite.moxiEntity.addLogic(new MoveRight());
