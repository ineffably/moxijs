    Class UICheckbox

    A checkbox control that can be checked or unchecked. Follows DOM API patterns for familiarity.

    Example

    const checkbox = new UICheckbox({
      checked: true,
      onChange: (checked) => console.log('Checked:', checked),
      size: 20
    });
