# Space Shooter 2 Spritesheet Documentation

This spritesheet contains various space-themed assets for creating space shooter games.

## Sprite Categories

### spaceAstronauts
Astronaut character sprites in various poses and orientations.

### spaceBuilding
Space station building modules and components including:
- Colored cargo containers (blue, yellow)
- Docking modules
- Connector pieces
- Station segments with windows

### spaceEffects
Visual effects including:
- Explosions
- Particle effects
- Energy bursts
- Weapon fire effects

### spaceMeteors
Various asteroid and meteor sprites:
- Large brown asteroids (polygonal shapes)
- Medium-sized space rocks
- Small debris pieces
- Different shapes and sizes for variety

### spaceMissiles
Projectile and missile sprites:
- Red-tipped missiles
- Energy projectiles
- Torpedoes
- Laser bolts
- Various colors (red, blue, green, orange)

### spaceParts
Miscellaneous space object components:
- Spheres and orbs (various colors and sizes)
- Geometric shapes
- Station components
- Connectors and joints
- Small detail elements

### spaceRocketParts
Individual rocket components for building custom rockets:
- Nose cones (red, gray, white)
- Fuel tanks
- Engine sections
- Fins and stabilizers
- Cockpit windows
- Thruster nozzles

### spaceRockets
Complete assembled rocket sprites:
- Classic red and white rockets
- Large launch vehicles
- Small scout rockets
- Multi-stage rockets with visible sections
- Rockets with different color schemes

### spaceShips
Various spacecraft designs:
- Red fighter ships with angular designs
- Gray utility vessels
- Small single-pilot craft
- Multi-engine ships
- Different orientations and views
- Combat-ready fighters with visible weapons

### spaceStation
Space station components and modules:
- Solar panel arrays (blue panels)
- Central hub modules
- Docking ports
- Communication arrays
- Habitat modules
- Antenna structures

## Usage Notes

- All sprites are designed with a consistent art style
- Colors include: red, blue, yellow, green, orange, gray, white, brown
- Sprites vary in size to represent different scales
- Many components can be combined to create custom ships and stations
- Suitable for 2D space shooter, strategy, or simulation games

## Asset Organization

The JSON file (`spaceShooter2_spritesheet.json`) contains frame data for all sprites including:
- Position (x, y coordinates)
- Size (width, height)
- Rotation data
- Trim information
- Source size data

Each sprite is named with its category prefix followed by a number (e.g., `spaceShips_001.png`)
